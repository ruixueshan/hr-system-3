DESIGN SPECIFICATION
====================
1. Purpose Statement: 
   HR管理系统的登录页面，用于验证用户身份并提供安全的访问入口。用户通过手机号和密码登录系统，进入各自权限的管理后台。需要体现专业、可信赖的企业形象，同时保持简洁高效的用户体验。

2. Aesthetic Direction: Luxury/refined
   - 专业、精致的视觉风格
   - 高品质的细节处理
   - 优雅的色彩搭配
   - 精心设计的排版和间距

3. Color Palette:
   - 主色调: #1a365d (深蓝，体现专业和稳定)
   - 辅色调: #d4af37 (金色，体现奢华和品质)
   - 背景色: #f8f9fa (浅灰，提供干净的基础)
   - 文字色: #2d3748 (深灰，确保可读性)
   - 强调色: #38b2ac (蓝绿色，用于交互反馈)

4. Typography:
   - 标题字体: 'Playfair Display', serif (优雅的衬线字体，用于品牌展示)
   - 正文字体: 'Source Sans Pro', sans-serif (清晰的无衬线字体，用于表单和说明)
   - 代码字体: 'SF Mono', 'Monaco', 'Menlo', monospace (技术相关展示)

5. Layout Strategy:
   - 不对称分割布局：左侧60%为登录表单区域，右侧40%为品牌展示区域
   - 品牌展示区域包含渐变背景、公司标识和品牌口号
   - 表单区域使用卡片设计，提升视觉层次
   - 精细的间距和微妙的阴影效果增强立体感
   - 响应式设计确保移动设备上的良好体验
